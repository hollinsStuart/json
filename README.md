# JSON

A json class by Hollins Yu.

## Installation

Make sure `google-test` and `google-benchmark` are installed.

## How to use

```c++
hollins::json jsonObject;
jsonObject.add("name", "John Doe");
jsonObject.add("age", 30);

jsonObject.prettyPrint();
```
